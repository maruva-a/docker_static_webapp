About this project

A simple static web application served from an Nginx container. The project is containerised with Docker and deployed via a CI/CD pipeline (Jenkins).

  • Static files: index.html, style.css
  • Image: nginx:alpine
  • Expose port: 80 (host port mapped by docker-compose)

Run locally (Docker)

docker build -t my-static-web .
docker run -d --name my-static-web -p 8081:80 my-static-web

Then open http://<server-ip>:8081

Using docker-compose

docker-compose up -d --build
docker-compose down

Compose file uses 80 inside the container; host port is configurable.

CI/CD (Jenkins)

Automated pipeline builds the image, stops old container and runs the new one.

  1. Checkout from GitHub
  2. docker build & docker run / docker-compose
  3. Verify deployment
